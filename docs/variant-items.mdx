---
title: Variant Items
---

<Callout type="info" title="Universal Support">
All endpoints support variant items.
</Callout>

Items with variants (Dopplers, Case Hardened, etc.) include a `variants` object in the response:

```json title="Variant Response"
{
  "market_hash_name": "â˜… Karambit | Doppler (Factory New)",
  "buff": { ... },
  "variants": {
    "Phase 1": {
      "display_name": "Phase 1",
      "version": "p1",
      "buff": { ... }
    },
    "Ruby": {
      "display_name": "Ruby",
      "version": "ruby",
      "buff": { ... }
    }
  }
}
```

## Doppler Phases

| Display Name | Version |
|--------------|---------|
| Phase 1-4 | `p1`, `p2`, `p3`, `p4` |
| Ruby | `ruby` |
| Sapphire | `sapphire` |
| Black Pearl | `blackpearl` |
| Emerald | `emerald` |

## Case Hardened Tiers

| Display Name | Version |
|--------------|---------|
| Tier 1-4 | `t1`, `t2`, `t3`, `t4` |
| Blue Gem | `singleblue` |
