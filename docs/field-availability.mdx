---
title: Marketplaces
description: See which price fields like listing price, buy orders, and volume are available from each CS2 marketplace in the cs2.sh API.
---

cs2.sh currently supports BUFF, YouPin, C5Game, CSFloat, Skinport, and Steam

| Marketplace | ask | bid | ask_volume | bid_volume | Additional      |
| ----------- | --- | --- | ---------- | ---------- | --------------- |
| BUFF        | Yes | Yes | Yes        | Yes        | -               |
| YouPin      | Yes | Yes | Yes        | Yes        | -               |
| C5Game      | Yes | Yes | Yes        | Yes        | -               |
| CSFloat     | Yes | -   | Yes        | -          | -               |
| Skinport    | Yes | -   | Yes        | -          | max/mean/median, history |
| Steam       | Yes | -   | Yes        | -          | -               |
