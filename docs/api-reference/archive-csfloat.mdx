title: Archive CSFloat
description: Access CSFloat historical sales records for CS2 items with the /v1/archive/csfloat endpoint. Individual sale prices, floats, and timestamps.
---

CSFloat daily sale averages — average price and volume per day. Available on Scale and Enterprise plans. Data spans ~2-3 years depending on the item. Updates ~once per day.

---

## POST /v1/archive/csfloat

Returns CSFloat daily sale history for up to 100 specific items. Available data range: ~2-3 years depending on the item. Supports variant items.

### Headers

```text title="Headers"
Authorization: Bearer YOUR_API_KEY
Accept-Encoding: gzip
Content-Type: application/json
```

### Request Body

<TypeTable
  type={{
    items: {
      type: "string[]",
      required: true,
      description: "List of market_hash_name values (max 100)",
    },
    start: {
      type: "string",
      required: false,
      description: "Start date (YYYY-MM-DD or RFC3339)",
      default: "2020-01-01",
    },
    end: {
      type: "string",
      required: false,
      description: "End date (YYYY-MM-DD or RFC3339)",
      default: "now",
    },
  }}
/>

  <Tab value="Go">

```go title="Go"
// [!code word:YOUR_API_KEY]
package main

import (
	"fmt"
	"io"
	"net/http"
	"strings"
	"time"
)

func main() {
	payload := `{"items":["AK-47 | Redline (Field-Tested)","★ Karambit | Doppler (Factory New)"],"start":"2024-01-01"}`

	req, err := http.NewRequest(
		http.MethodPost,
		"https://api.cs2.sh/v1/archive/csfloat", // [!code highlight]
		strings.NewReader(payload),
	)
	if err != nil {
		panic(err)
	}

	req.Header.Set("Authorization", "Bearer YOUR_API_KEY") // [!code highlight]
	req.Header.Set("Accept-Encoding", "gzip")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{Timeout: 30 * time.Second}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(body))
}
```
