title: Price History
description: Query historical OHLC price data for CS2 items with the /v1/prices/history endpoint. Daily and hourly candles with marketplace filtering.
---

<Callout type="info" title="Available on Scale and Enterprise plans">
  See [Plans](/docs/pricing) for details.
</Callout>

Returns historical prices with OHLC (Open/High/Low/Close) aggregation. Recommended for recent, high-resolution historical data.

- **Open**: First price recorded in the interval
- **High**: Highest price during the interval
- **Low**: Lowest price during the interval
- **Close**: Last price recorded in the interval

Available data range: December 24th, 2025 onwards.

<Callout type="info" title="Update Frequency">
Updated continuously, at the same rate as [Latest](/docs/api-reference/prices-latest). For long-term historical data, use [Archive History](/docs/api-reference/archive-history) instead.
</Callout>

**Intervals and maximum ranges:**

| Interval | Max Range |
|----------|-----------|
| `5m` | 14 days |
| `30m` | 90 days |
| `1h` | 180 days |
| `1d` | 365 days |

<Callout type="info" title="Need More History?">
For longer-term historical data, use [Archive History](/docs/api-reference/archive-history) instead.
</Callout>

## POST /v1/prices/history

Returns historical prices with OHLC aggregation for up to 100 specific items. Available on Scale and Enterprise plans only.

### Headers

```text title="Headers"
Authorization: Bearer YOUR_API_KEY
Accept-Encoding: gzip
Content-Type: application/json
```

### Request Body

<TypeTable
  type={{
    items: {
      type: "string[]",
      required: true,
      description: "List of market_hash_name values (max 100)",
    },
    start: {
      type: "string",
      required: true,
      description: "Start date (YYYY-MM-DD or RFC3339)",
    },
    end: {
      type: "string",
      required: false,
      description: "End date (YYYY-MM-DD or RFC3339)",
      default: "now",
    },
    sources: {
      type: "string[]",
      required: false,
      description: "Filter to specific sources. Values: buff, youpin, csfloat, skinport, steam, c5game",
      default: "all",
    },
    interval: {
      type: "string",
      required: false,
      description: "Aggregation interval. Values: 5m, 30m, 1h, 1d",
      default: "5m",
    },
  }}
/>

  <Tab value="Go">

```go title="Go"
// [!code word:YOUR_API_KEY]
package main

import (
	"fmt"
	"io"
	"net/http"
	"strings"
	"time"
)

func main() {
	payload := `{"items":["AK-47 | Redline (Field-Tested)","★ Karambit | Doppler (Factory New)"],"start":"2026-01-01","interval":"1h","sources":["buff","csfloat"]}`

	req, err := http.NewRequest(
		http.MethodPost,
		"https://api.cs2.sh/v1/prices/history", // [!code highlight]
		strings.NewReader(payload),
	)
	if err != nil {
		panic(err)
	}

	req.Header.Set("Authorization", "Bearer YOUR_API_KEY") // [!code highlight]
	req.Header.Set("Accept-Encoding", "gzip")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{Timeout: 30 * time.Second}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(body))
}
```
