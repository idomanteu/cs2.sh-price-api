title: Archive History
description: Query long-term CS2 pricing archives per marketplace with the /v1/archive/history endpoint. Multi-year historical price data.
---

<Callout type="info" title="Available on Scale and Enterprise plans">
  See [Plans](/docs/pricing) for details.
</Callout>

Long-term historical price data per source. Stores comprehensive prices across BUFF, YouPin, and C5Game, plus a precomputed "aggregate" for overall prices.

Available data range: ~2-3 years, depending on the item. Supports variant items.

<Callout type="info" title="Update Frequency">
Data updates ~1-2 times per day. For recent data, use [Price History](/docs/api-reference/prices-history) instead.
</Callout>

**Sources:** `aggregate` (default), `buff`, `youpin`, `c5game`

**Intervals:** `1h` (hourly), `1d` (daily, default)

## POST /v1/archive/history

Returns long-term archive price history for up to 100 specific items. Available on Scale and Enterprise plans only.

### Headers

```text title="Headers"
Authorization: Bearer YOUR_API_KEY
Accept-Encoding: gzip
Content-Type: application/json
```

### Request Body

<TypeTable
  type={{
    items: {
      type: "string[]",
      required: true,
      description: "List of market_hash_name values (max 100)",
    },
    start: {
      type: "string",
      required: true,
      description: "Start date (YYYY-MM-DD or RFC3339)",
    },
    end: {
      type: "string",
      required: false,
      description: "End date (YYYY-MM-DD or RFC3339)",
      default: "now",
    },
    sources: {
      type: "string[]",
      required: false,
      description: "Filter to specific sources. Values: aggregate, buff, youpin, c5game",
      default: "aggregate",
    },
    interval: {
      type: "string",
      required: false,
      description: "Aggregation interval. Values: 1h, 1d",
      default: "1d",
    },
  }}
/>

  <Tab value="Go">

```go title="Go"
// [!code word:YOUR_API_KEY]
package main

import (
	"fmt"
	"io"
	"net/http"
	"strings"
	"time"
)

func main() {
	payload := `{"items":["AK-47 | Redline (Field-Tested)","★ Karambit | Doppler (Factory New)"],"start":"2025-01-01","sources":["aggregate"],"interval":"1d"}`

	req, err := http.NewRequest(
		http.MethodPost,
		"https://api.cs2.sh/v1/archive/history", // [!code highlight]
		strings.NewReader(payload),
	)
	if err != nil {
		panic(err)
	}

	req.Header.Set("Authorization", "Bearer YOUR_API_KEY") // [!code highlight]
	req.Header.Set("Accept-Encoding", "gzip")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{Timeout: 30 * time.Second}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(body))
}
```
