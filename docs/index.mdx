cs2.sh is an API platform that provides Counter-Strike 2 price data through a REST API. The system collects data from independent marketplaces every 3-5 minutes. All data is refreshed as soon as it's available.

Supported Marketplaces:

BUFF, YouPin, C5Game, CSFloat, Skinport, Steam

**ask** = Lowest listing price (what you'd pay to buy now). **bid** = Highest buy order (what you'd receive if selling now). All prices in USD.

## Quick Start

<Steps>
<Step>

### Create an account

[Sign up](https://cs2.sh/signup) and subscribe to a [plan](/docs/pricing).

</Step>
<Step>

### Get your API key

Generate an API key from your [dashboard](https://cs2.sh/dashboard).

</Step>
<Step>

### Make your first request

All authenticated requests require `Accept-Encoding: gzip`.

<Tabs items={["cURL", "Python", "Node.js", "Go"]} groupId="request-language" persist updateAnchor>
  <Tab value="cURL">

```bash title="cURL"
# [!code word:YOUR_API_KEY]
curl -X GET "https://api.cs2.sh/v1/prices/latest" \ # [!code highlight]
  -H "Authorization: Bearer YOUR_API_KEY" \ # [!code highlight]
  -H "Accept-Encoding: gzip"
```

  </Tab>
  <Tab value="Python">

```python title="Python"
# [!code word:YOUR_API_KEY]
import requests

response = requests.get(
    "https://api.cs2.sh/v1/prices/latest",  # [!code highlight]
    headers={
        "Authorization": "Bearer YOUR_API_KEY",  # [!code highlight]
        "Accept-Encoding": "gzip",
    },
    timeout=30,
)

response.raise_for_status()
print(response.json())
```

  </Tab>
  <Tab value="Node.js">

```js title="Node.js"
// [!code word:YOUR_API_KEY]
async function main() {
  const response = await fetch("https://api.cs2.sh/v1/prices/latest", { // [!code highlight]
    method: "GET",
    headers: {
      Authorization: "Bearer YOUR_API_KEY", // [!code highlight]
      "Accept-Encoding": "gzip",
    },
  });

  if (!response.ok) {
    throw new Error(`Request failed: ${response.status} ${response.statusText}`);
  }

  const data = await response.json();
  console.log(data);
}

main().catch(console.error);
```

  </Tab>
  <Tab value="Go">

```go title="Go"
// [!code word:YOUR_API_KEY]
package main

import (
	"fmt"
	"io"
	"net/http"
	"time"
)

func main() {
	req, err := http.NewRequest(http.MethodGet, "https://api.cs2.sh/v1/prices/latest", nil) // [!code highlight]
	if err != nil {
		panic(err)
	}

	req.Header.Set("Authorization", "Bearer YOUR_API_KEY") // [!code highlight]
	req.Header.Set("Accept-Encoding", "gzip")

	client := &http.Client{Timeout: 30 * time.Second}
	resp, err := client.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(body))
}
```

## Endpoints

| I need... | Endpoint | Plan |
|---|---|---|
| Current prices (all items) | `GET /v1/prices/latest` | Developer+ |
| Current prices (specific items) | `POST /v1/prices/latest` | Developer+ |
| Recent price history (OHLC) | `POST /v1/prices/history` | Scale+ |
| Long-term price history | `POST /v1/archive/history` | Scale+ |
| CSFloat sales history | `POST /v1/archive/csfloat` | Scale+ |
| Liquidity scores | `GET\|POST /v1/liquidity` | Scale+ |
| Service health | `GET /health` | Free |
